for...of vs. for...in

Today, I was just curious how many words test your vocab would give me if I select all the checkboxes. But I was too lazy to check them all manually, especially on the 2/3 step. Thus I tried to execute some simple JavaScript code in my Firefox web console, in order to select them in a batch and ended up in the following code. 

for (let elem in document.querySelectorAll('input[type="checkbox"]') {
    if (!elem.checked) {
        elem.checked = true
    }
}

But nothing happened and I googled to find out why. According to MDN, for...in iterates over properties of an object whereas for...of iterates over each element of a collection. Therefore, the correct version should be the following. 

for (let elem of document.querySelectorAll('input[type="checkbox"]') {
    if (!elem.checked) {
        elem.checked = true
    }
}
